# Frontend Resume

一份以 HTML、CSS 和 JavaScript 製作的個人履歷頁，主打乾淨的視覺呈現與簡單互動。可直接在瀏覽器開啟，不需要額外安裝套件。

## 內容特色

- 響應式版面，支援桌機與手機瀏覽
- 深色 / 淺色模式切換
- 多語系切換介面
- 履歷區塊包含個人簡介、學歷、經歷、技能與語言能力
- 使用本地圖片資源，不依賴外部服務

## 使用方式

直接開啟：

```text
resume.html
```

或將整個專案資料夾放到靜態網站服務中部署即可。

## 專案結構

```text
.
├── resume.html
├── README.md
└── assets/
    ├── profile-photo.png
    ├── NUTC_logo.gif
    └── asia-site-icon.png
```

## 自訂內容

- 履歷文字、語言內容與互動邏輯都在 `resume.html`
- 個人照片與學校圖示放在 `assets/`
- 樣式集中在 `resume.html` 內的 `<style>` 區塊

## 技術

- HTML
- CSS
- JavaScript
